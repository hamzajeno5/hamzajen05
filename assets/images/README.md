# Images Folder

This folder is intended for storing images for the Glaz Casablanca website.

## Recommended Images

To enhance the website, consider adding the following images:

1. **Hero Background** (`hero-bg.jpg`) - A stunning photo of the restaurant's interior or exterior
2. **Restaurant Interior** (`interior-1.jpg`, `interior-2.jpg`) - Showcase the contemporary design and teal accents
3. **Cocktails** (`cocktail-1.jpg`, `cocktail-2.jpg`, `cocktail-3.jpg`) - Professional photos of your signature cocktails
4. **Food Dishes** (`dish-1.jpg`, `dish-2.jpg`, `dish-3.jpg`) - Mouthwatering photos of your gourmet meals
5. **Ambiance** (`ambiance-1.jpg`, `ambiance-2.jpg`) - Photos capturing the vibrant atmosphere
6. **Logo** (`logo.png`, `logo.svg`) - Restaurant logo in various formats

## Image Specifications

- **Format**: JPG for photos, PNG for graphics with transparency, SVG for logos
- **Resolution**: Minimum 1920x1080 for hero images, 800x600 for gallery items
- **Optimization**: Compress images for web use to maintain fast loading times
- **Naming**: Use descriptive names with hyphens (e.g., `teal-interior-design.jpg`)

## Usage

Once images are added to this folder, update the HTML file to replace the placeholder elements with actual images:

```html
<!-- Replace placeholder with: -->
<img src="assets/images/hero-bg.jpg" alt="Glaz Casablanca Interior">
```
